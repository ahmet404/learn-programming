## Apa itu Version Control System?
**Version Control System** disebut juga **revision control system** atau **source code management** adalah sebuah sistem yang mengelola dan merekam setiap perubahan yang terjadi dari sebuah berkas atau dokumen, program komputer, website dan kumpulan informasi lain yang dibuat secara digital dari waktu ke waktu sehingga dapat dilihaf kembali versi khusu suatu saat nanti.

## Kegunaan Version Control System
- Sebuah system yang menyimpan \'rekaman / snapshot' perubahan pada source code.
- Memungkinkan bekerja berkolaborasi dengan lebih baik.
- Mengetahui siapa yang melakukan dan kapan sebuah perubahan terjadi.
- Memungkinkan kita untuk kembali ke keadaan sebelum perubahan (checkout).
