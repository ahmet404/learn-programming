## Apa itu Version Control System?
**Version Control System** disebut juga **revision control system** atau **source code management** adalah system yang mengelola perubahan dari sebuah dokumen, program komputer, website dan kumpulan informasi lain.

## Kegunaan Version Control System
- Sebuah system yang menyimpan \'rekaman / snapshot' perubahan pada source code.
- Memungkinkan bekerja berkolaborasi dengan lebih baik.
- Mengetahui siapa yang melakukan dan kapan sebuah perubahan terjadi.
- Memungkinkan kita untuk kembali ke keadaan sebelum perubahan (checkout).
